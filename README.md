📊 Trend vs Review Rating Analysis

– 수요 증가가 사용자 경험에 미치는 영향 분석

13만+ 리뷰 데이터를 기반으로
트렌드(수요) 상승이 사용자 경험 지표(평점)에 미치는 영향을 검증하고,
기능 추가가 아닌 경험 순서 조정이 핵심 해결책임을 도출한 데이터 기반 PM 프로젝트

1. 프로젝트 개요 (Overview)

본 프로젝트는 리뷰 13만 건과 주별 트렌드 지수 데이터를 활용해
외부 수요 변화가 사용자 경험 품질(리뷰 평점)에 어떤 영향을 미치는지 분석하는 것을 목표로 합니다.

단순 상관관계 분석이 아니라,

언제 평점이 하락하는지

어떤 조건에서 하락이 심해지는지

PM 관점에서 무엇을 바꿔야 하는지

를 의사결정 가능한 인사이트로 연결하는 데 집중했습니다.

2. 문제 정의 (Problem Statement)
관측된 현상 (Fact)

주별 리뷰 수가 급증하는 시점에 평균 평점이 급격히 하락

동일 시점에 트렌드 지수 또한 상승 구간에 위치

왜 문제인가 (Why it matters)

리뷰 평점은 전환, 재주문, 리텐션에 직접적인 영향을 미치는 핵심 경험 지표

트렌드 상승이 곧바로 평점 하락으로 이어질 경우
성장 성과가 장기 사용자 가치(LTV)를 훼손할 위험 존재

3. 목표 및 가설 (Goal & Hypothesis)
🎯 목표

주별 트렌드 지수와 평균 평점 간의 관계를 정량적으로 검증

평점 하락이 발생하는 구조적 원인을 설명

🧠 가설

H1. 트렌드 지수가 상승할수록 평균 평점은 하락할 것이다

H2. 리뷰 수가 급증하는 주차에서 평점 하락 폭은 더 커질 것이다
